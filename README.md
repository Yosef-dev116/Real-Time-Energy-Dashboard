# Real-Time Energy Dashboard

Full-stack web application with a Python backend and a React frontend.

## Project Structure
- `Backend/` – Python backend API
- `frontend/` – React frontend application

## Features
- Backend API for handling energy data
- Frontend dashboard for displaying data
- Separation of frontend and backend logic

## Technologies Used
**Backend**
- Python
- FastAPI

**Frontend**
- React
- JavaScript
- CSS
- 
## Demo
This project does not have a hosted live demo.

The frontend and backend are intended to be run locally for development and testing.

## How to Run Locally

### Backend
1. Navigate to the `Backend` folder
2. Install dependencies
3. Run the backend server

### Frontend
1. Navigate to the `frontend` folder
2. Install dependencies
3. Start the development server
